Portfolio Website (RWD; 2019)

Objective:

Building a portfolio website for an abstract painter who wished to reach a wider audience by sharing her work online.
I put the main focus on assuring that the web content would respond and adjust when the user switches from their laptop to iPad.

Process:

With these goals in mind, I designed a website – proposing the color palette, the logo (Adobe Illustrator), the layout, and the pictures (edited in Adobe Lightroom Classic). Next months included coding the separate pages (mainly plain html, css, js), consulting with the client, and making sure the website looks good on the screens of desktop computers as well as mobile devices.

The Mole Game (JS; 2020)

Objective:

Building a JS Feed-a-Mole game.

Process:

Scripting in plain html, css, js to fulfill the following user stories:

  1. Make the hungry moles and Royal moles (rare) show up and disappear randomly.
  2. Make the hungry moles clickable - 1 point for a successful click
  3. Make clicking the Royal Moles worth twice as many points
  4. Display a score meter
  5. Add fed moles - clicked, sad moles - (not clicked), mole butts - just before leaving
  6. After gaining 10 points - successful clicks show the Win Screen
